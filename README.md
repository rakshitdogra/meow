## рЯРЊ Introduction

**MEOW** is a dynamically-typed, interpreted programming language designed entirely around cat behavior and vocabulary. Every keyword, construct, and concept maps to something a cat would do. Variables *purr* into existence. Logic *hisses*. Loops *stalk* their prey. Functions *claw* through logic. 

At the core of MEOW is the sacred `meow` keyword вАФ the primary output command. Everything in the language revolves around it, just as everything in a house revolves around the cat.

---

## вЪ° Features

- **Cat-Powered Syntax**: A fun, esoteric-style language that remains fully functional and practical to write.
- **Dynamic Typing**: No explicit type declarations; types are resolved at runtime.
- **First-class Functions**: Functions can capture their environment and be passed around.
- **Built-in Collections**: Built-in support for arrays (called *Litters*).
- **Rich Standard Library**: Includes over 14 built-in utility functions for math, strings, and arrays.
- **Browser-Based IDE**: Includes a comprehensive web-based IDE built entirely with Vanilla technologies.

## вЬПпЄП Syntax & Core Concepts

### Keywords
- `purr` вАФ Variable declaration
- `meow` вАФ Print to standard output
- `hiss` вАФ `if` condition
- `stalk` вАФ `while` loop
- `claw` вАФ Function definition
- `pawsitive` вАФ Boolean `true`
- `negative` вАФ Boolean `false`
- `nap` вАФ `null` equivalent

### Quick Example


## рЯТї MEOW IDE v2

The project comes with a fully-featured, browser-based Integrated Development Environment (`meow-ide.html`).
